2. Motion/Navigation
3. Shortcuts
4. Save/Quit
5. Tabs/Split
6. Visual Commands
7. Macros
8. Substitution/Searching
9. Colorschemes
10. Marks
11. Other commands
12. Set options
13. Buffers
14. Line Folding
